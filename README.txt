Instructions to publish on GitHub Pages:
1. Create a new repository on GitHub (e.g., jannatul-portfolio).
2. Upload these files to the repository root (or push via git).
3. Add your profile image file named `profile.jpg` to the same repo root to appear on the homepage.
4. In repo settings -> Pages -> select branch 'main' and folder '/' then Save.
5. Your site will be available at https://<username>.github.io/<repo-name> (or username.github.io for root repo).

Edit content by opening the HTML files and updating text (publications, projects, links).
